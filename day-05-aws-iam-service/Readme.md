# AWS Identity and Access Management (IAM) service

## 01. IAM Users

- An IAM User is a distinct entity created within an AWS account to represent a specific _person_ or _application_.
- Broadly categorized into two types:
  - Root Users
  - IAM Users

### Create an IAM User

- Sign in to the AWS Management Console and navigate to the IAM service.
- Select **Users** and click **Create user**
  - Enter a **Username** and choose whether to grant console access.
  - Set permissions by adding the user to a group, copying permissions from an existing user, or attaching policies directly.
  - Review and create the user, saving credentials for the user.
