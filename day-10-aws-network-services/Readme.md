# AWS Network Services

## 01. `Virtual Private Cloud (VPC)`

### `VPC` Overview

- A AWS Virtual Private Cloud (VPC) is a logically isolated, dedicated network service, providing security and control over IP ranges, subnets, and routing.
- Key concepts include CIDR blocks for IP range definition, Subnets for segmenting resources across zones, and Route Tables for traffic flow.

### `VPC` Key Concepts

- IP Addressing
- CIDR Blocks
- Subnets
- Route Tables
- Internet Gateway
- NAT Gateway
- Security Groups
