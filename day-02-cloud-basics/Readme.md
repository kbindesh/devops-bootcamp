# Cloud Basics

## 01. Cloud Computing Overview

- _Cloud computing_ provides on-demand access to virtualized IT resources such as servers, storage, databases, and software, over the internet via a pay-as-you-go model.

- It allows businesses to avoid upfront hardware costs (OpEx instead of CapEx), scale rapidly, and manage infrastructure efficiently.
- Core components include virtualization, data centers, and networking resources.

## 02. Cloud Computing - Key Characterstics

- On-Demand Self-Service

- Broad Network Access
- Resource Pooling
- Rapid Elasticity
- Measured Service

## 03. Cloud `Service Models`

1. IaaS (Infrastructure as a Service)
   - It provides raw computing, storage, and networking resources over the internet as a service.

2. PaaS (Platform as a Service)
   - It provides computing, storage, and networking resource and software tools in managed form for the application development.

3. SaaS (Software as a Service)
   - Delivers fully managed software applications over the internet.

## 04. Cloud `Deployment Models`

1. Public Cloud
   - Services offered over the public internet by providers like AWS, Microsoft Azure, and Google Cloud.

2. Private Cloud
   - Dedicated infrastructure for a single organization, offering higher security and control.

3. Hybrid Cloud
   - A combination of public and private clouds, allowing data and applications to be shared between them.
