# AWS `Storage` Services

## Different type of Storage and AWS offerings

- **Object storage**
  - Amazon S3

- **Block storage**
  - Amazon EBS
- **File storage**
  - EFS
  - FSx
- **Archive storage**
  - Amazon S3 Glacier
- **Relational Data storage**
  - RDS
  - Aurora

- **Data Warehousing storage**
  - Amazon RedShift
