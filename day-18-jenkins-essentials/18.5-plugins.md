# Managing Jenkins Plugins

- `Jenkins plugins` are add-ons that extend the core functionality of the Jenkins automation server.

- Jenkins has over 1,800 free plugins that are available in the official repository, they allow Jenkins to integrate with virtually any tool in the DevOps ecosystem, including source control, build tools, and cloud providers.

- The **Jenkins Plugin Manager** is the primary interface for administrators to install, update, and manage the numerous plugins that extend the core functionality of a Jenkins environment.

- Plugins integrate Jenkins with a wide variety of build tools, cloud providers, and analysis tools, with over a thousand available in the official **Jenkins Plugins index**.

- Jenkins Plugins index - https://plugins.jenkins.io/

- Commonly used Plugins
  - Git
  - Pipeline
  - Role-based Authorization
  - Docker
  - Kubernetes
  - Email Extension
  - Maven Integration
  - SonarQube
  - Pipeline: Stage View

## Lab: Installing a Jenkins Plugin

1. Navigate to Jenkins Dashboard >> **Manage Jenkins** >> **Plugins**.

2. Under the **Available** tab, plugins available for download from the configured marketplace can be searched and installed.

3. Select appropriate plugin(s), click on **Install** button

## Lab: Removing a Jenkins Plugin

1. To uninstall a plugin, navigate to Jenkins Dashboard >> **Manage Jenkins** >> **Plugins**

2. Now select the **Installed** tab on the Plugins page >> Select plugin(s) to be uninstalled >> Uninstall
